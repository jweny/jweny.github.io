给Blog换个皮肤

0x01 碎碎念

大壮突然想没事写点东西，让我给搭建个blog。

这么简单的需求，心想直接hexo+github Page就搞定了。奈何hexo自带的主题越看越丑，所以在github上搜一搜，发现 fluid主题 还挺帅的，就折腾了俩小时，这里简单记录下。

0x02 安装

hexo的安装方式已经一搜一大堆了，类似 这种 。

简单来说，基于Node环境

npm install -g hexo-cli
npm install -g hexo

新建一个文件夹，用于保存blog项目，切换到该文件夹

hexo init
npm install

安装fluid主题

npm install --save hexo-theme-fluid
// 引入本地文件的插件
npm install https://github.com/CodeFalling/hexo-asset-image --save

下载一份 [配置文件] (https://github.com/fluid-dev/hexo-theme-fluid/blob/master/_config.yml)

命名为_config.fluid.yml

依据教程修改配置即可。
